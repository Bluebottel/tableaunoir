Welcome to Tableau noir, the coolest online blackboard. Press Esc. for help.

Tableau noir - Menu

My keyboard

Esc. = show/hide this menu
e = switch between chalk and eraser
c = switch the color
d = divide the screen

My magnets

Suppr = delete the current magnet
m = transform the selected area in a magnet (to select simply draw a contour around the area with your chalk) / flat the current magnet to the blackboard
p = print the current magnet (like rubber stamping)

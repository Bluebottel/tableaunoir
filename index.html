Welcome to Tableaunoir

Tableaunoir is the greatest (!) online blackboard. If it does not work, please update your browser. Work best on Firefox, Chrome or Chromium, last versions. Of course, you can draw and erase, but you can also create magnets (think of them as fridge magnets) for making interactive courses, presentations and discussions with students.

The project is open source and the source code is available at:
https://github.com/tableaunoir

The button-free mode does not show any button, so that your students will not be distracted by any graphical user interface elements. Press Esc. or F1 for show the menu that explains all the keyboard shortcuts, and give some help.
The button mode offers few buttons for changing the colors, for erasing and for showing the menu.
